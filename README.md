# topdownshooter

A top down shooter based on zombies and other shooters.

## Features:

- Customizable spawn waves
- Points integration
- Game over mechanics and lives
- resolution scaling for 4:3 windowed screens
- start screen and game over screen

## Weapons:
- reg bullets
- axe boomerang
